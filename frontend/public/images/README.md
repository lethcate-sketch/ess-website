# 写真・ロゴの差し替え方法

写真・ロゴは **管理ダッシュボードから差し替えられます**（コードを触る必要はありません）。

## ① 管理ダッシュボードから差し替える（おすすめ）

1. 管理者でログイン → **管理** メニューを開く
2. 画像の場所ごとに次のページで設定します:
   - **トップページの画像（ロゴ・ヒーロー・ギャラリー）** → 「概要」ページの「サイト画像（トップページ）」
   - **サークル紹介ページの画像（カバー・活動写真）** → 「サークル紹介」ページの「ページ画像」
3. 各画像で、次のどちらかで差し替え:
   - **ファイルをアップロード**：PCから画像を選ぶだけ（大きな写真は自動で軽量化されます）
   - **URLを貼り付け**：画像URL（`https://…`）を入れて「適用」
4. 「デフォルトに戻す」で初期画像に戻せます。
5. 反映には数十秒かかる場合があります（表示が古いときはブラウザを再読み込み）。

> ロゴは **背景が透明な PNG** を推奨します（ヘッダーは明るい背景のため）。

## ② コードで初期値を変える（上級者向け）

管理画面で未設定のときに使われる「デフォルト画像」を変えたい場合:

1. 画像をこのフォルダ `frontend/public/images/` に置く（例: `hero.jpg`）
2. `frontend/lib/siteImages.ts` の `IMAGE_DEFAULTS` の該当 `src` をローカルパスに変更:
   ```ts
   hero: "https://picsum.photos/seed/ess-hero/1200/900",
   //   ↓
   hero: "/images/hero.jpg",
   ```
3. ロゴの初期値は `IMAGE_DEFAULTS.logo`（既定 `/images/logo.svg`）。
4. コミットして push すると Render が自動反映します。

## 仕組み（参考）
- 公開ページは `/api/images/<キー>` を参照します。
- そのキーに管理画面で設定した画像があればそれを、無ければ `IMAGE_DEFAULTS` の画像を配信します。
- アップロード画像は DB に保存され、再デプロイしても保持されます。

## 推奨サイズ・形式
- 形式: 写真=JPG、ロゴ=透過PNG
- ヒーロー≈1600×1000px / 活動写真≈1000×750px
- アップロードは自動で最大1600px・JPEG品質85%に軽量化されます
